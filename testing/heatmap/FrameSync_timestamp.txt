## 1. Synchronization Concepts

### Why Monotonic Timestamps?
- **Clock Stability**: Monotonic clocks don't jump backward (unlike system clocks)
- **Precision**: High-resolution timing for frame-level accuracy
- **Cross-platform**: Available on Linux, macOS, and Windows
- **Real-time Friendly**: Designed for timing-sensitive applications

### Why Ring Buffers?
- **Bounded Memory**: Prevents memory growth during long runs
- **Efficient Access**: O(1) insertion and removal
- **Natural Ordering**: Maintains temporal sequence
- **Overflow Handling**: Automatically discards old frames

### Synchronization Strategy
1. Capture both audio and video frames with monotonic timestamps
2. Store frames in ring buffers with timestamps
3. Find best matching pairs within tolerance window
4. Process synchronized frames together

## 2. Core Synchronization Classes

```python
import time
import threading
from collections import deque
import cv2
import numpy as np

class AudioVideoSynchronizer:
    """
    Synchronizes audio and video frames using monotonic timestamps.
    
    This class maintains separate ring buffers for audio and video frames,
    each with their timestamps, and provides methods to find the best
    matching pairs within a specified tolerance window.
    """
    
    def __init__(self, buffer_size=10, sync_tolerance=0.033):
        """
        Initialize the synchronizer.
        
        Args:
            buffer_size: Maximum number of frames to keep in each buffer
            sync_tolerance: Maximum time difference (seconds) for frame matching
        """
        self.audio_buffer = deque(maxlen=buffer_size)
        self.video_buffer = deque(maxlen=buffer_size)
        self.sync_tolerance = sync_tolerance  # ~1 frame at 30fps
        self.lock = threading.Lock()  # Thread safety for multi-threaded access
        
    def add_audio_frame(self, audio_data, timestamp=None):
        """
        Add an audio frame with timestamp to the buffer.
        
        Args:
            audio_data: Audio frame data (numpy array)
            timestamp: Monotonic timestamp (auto-generated if None)
        """
        if timestamp is None:
            timestamp = time.monotonic()
            
        with self.lock:
            self.audio_buffer.append((timestamp, audio_data))
        
    def add_video_frame(self, video_frame, timestamp=None):
        """
        Add a video frame with timestamp to the buffer.
        
        Args:
            video_frame: Video frame (numpy array)
            timestamp: Monotonic timestamp (auto-generated if None)
        """
        if timestamp is None:
            timestamp = time.monotonic()
            
        with self.lock:
            self.video_buffer.append((timestamp, video_frame))
        
    def get_synchronized_pair(self):
        """
        Find the best matching audio-video pair within tolerance.
        
        Returns:
            tuple: (audio_data, video_frame) or (None, None) if no match
        """
        with self.lock:
            if not self.audio_buffer or not self.video_buffer:
                return None, None
                
            best_audio = None
            best_video = None
            best_diff = float('inf')
            
            # Find closest timestamp match
            for a_ts, a_data in self.audio_buffer:
                for v_ts, v_frame in self.video_buffer:
                    time_diff = abs(a_ts - v_ts)
                    if time_diff < self.sync_tolerance and time_diff < best_diff:
                        best_diff = time_diff
                        best_audio = a_data
                        best_video = v_frame
                        
            return best_audio, best_video
    
    def get_buffer_status(self):
        """Get current buffer sizes for monitoring."""
        with self.lock:
            return len(self.audio_buffer), len(self.video_buffer)
    
    def clear_buffers(self):
        """Clear all buffers (useful for reset)."""
        with self.lock:
            self.audio_buffer.clear()
            self.video_buffer.clear()
```

## 3. Ring Buffer Implementation

```python
class FrameBuffer:
    """
    Ring buffer for storing frames with timestamps.
    
    This class provides efficient storage and retrieval of frames
    with their associated timestamps, using a bounded circular buffer.
    """
    
    def __init__(self, max_size=30):
        """
        Initialize the frame buffer.
        
        Args:
            max_size: Maximum number of frames to store (default: 1 second at 30fps)
        """
        self.buffer = deque(maxlen=max_size)
        self.timestamps = deque(maxlen=max_size)
        self.lock = threading.Lock()
        
    def add_frame(self, frame, timestamp=None):
        """
        Add a frame with timestamp to the buffer.
        
        Args:
            frame: Frame data (numpy array)
            timestamp: Monotonic timestamp (auto-generated if None)
        """
        if timestamp is None:
            timestamp = time.monotonic()
            
        with self.lock:
            self.buffer.append(frame.copy())  # Copy to avoid reference issues
            self.timestamps.append(timestamp)
        
    def get_frame_at_time(self, target_time, tolerance=0.016):
        """
        Get frame closest to target timestamp.
        
        Args:
            target_time: Target timestamp to match
            tolerance: Maximum time difference for matching (default: ~0.5 frame)
            
        Returns:
            numpy array: Closest frame or None if no match within tolerance
        """
        with self.lock:
            if not self.buffer:
                return None
                
            best_frame = None
            best_diff = float('inf')
            
            for frame, ts in zip(self.buffer, self.timestamps):
                diff = abs(ts - target_time)
                if diff < best_diff and diff < tolerance:
                    best_diff = diff
                    best_frame = frame
                    
            return best_frame
    
    def get_latest_frame(self):
        """Get the most recent frame."""
        with self.lock:
            return self.buffer[-1] if self.buffer else None
    
    def get_oldest_frame(self):
        """Get the oldest frame."""
        with self.lock:
            return self.buffer[0] if self.buffer else None
    
    def get_buffer_size(self):
        """Get current buffer size."""
        with self.lock:
            return len(self.buffer)
    
    def clear(self):
        """Clear all frames from buffer."""
        with self.lock:
            self.buffer.clear()
            self.timestamps.clear()
```

## 6. Performance Considerations

### Memory Management
- **Buffer Sizing**: Choose buffer sizes based on expected latency and memory constraints
- **Frame Copying**: Copy frames to avoid reference issues in multi-threaded environments
- **Garbage Collection**: Monitor memory usage during long runs

### Timing Precision
- **Tolerance Tuning**: Adjust sync tolerance based on system characteristics
- **Clock Drift**: Monitor for clock drift between audio and video sources
- **Latency Compensation**: Account for processing delays in timestamp calculations

### Thread Safety
- **Lock Usage**: Use locks for thread-safe buffer access
- **Producer-Consumer**: Consider separate threads for audio and video capture
- **Deadlock Prevention**: Avoid nested locks and long critical sections

### Real-time Constraints
- **Priority Scheduling**: Use real-time scheduling for critical threads
- **CPU Affinity**: Pin threads to specific CPU cores for consistent performance
- **Interrupt Handling**: Handle system interrupts gracefully
